# Internet of Things (IoT) Projects

This repository contains my IoT projects developed using Raspberry Pi Pico W and simulated using Wokwi. My work mainly focuses on building and testing different embedded system projects using MicroPython, where I explore how code interacts with hardware components in a practical way.

# 
Through these projects, I have worked on multiple concepts such as controlling onboard components, understanding basic input/output operations, and experimenting with different virtual components and circuits. Instead of using physical hardware, I used Wokwi to design, test, and debug my projects in a virtual environment.

This approach is useful because it allows learning and experimentation without needing physical components, making it easier to understand IoT concepts, test ideas quickly, and build a strong foundation in embedded systems. It also helps in identifying and fixing errors in code before implementing them on real hardware.
